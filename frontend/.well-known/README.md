# Apple Pay Domain Verification

Этот файл будет предоставлен Stripe Dashboard при регистрации домена.

## Как получить файл:

1. Откройте https://dashboard.stripe.com/settings/payment_methods
2. Найдите раздел "Apple Pay"
3. Нажмите "Add new domain"
4. Введите ваш домен (например: phonecleanerplus.info)
5. Stripe предоставит файл для скачивания
6. Скачайте и поместите файл в эту директорию с именем:
   `apple-developer-merchantid-domain-association`

После размещения файла нажмите "Verify" в Stripe Dashboard.
